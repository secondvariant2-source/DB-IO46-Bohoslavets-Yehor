### Міграція 1: Додавання таблиці відгуків
**Опис**: Створено нову модель `Review` для зберігання оцінок та коментарів читачів до книг.
* **Код у schema.prisma**:
```prisma
model Review {
  id        Int      @id @default(autoincrement())
  rating    Int
  comment   String?
  bookId    Int
  book      Book     @relation(fields: [bookId], references: [id])
}
